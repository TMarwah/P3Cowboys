Robot Class

The class is created and called Robot, with functions to set and get attributes

class Robot:
    def setName(self,n):
        self._name=n
    def getName(self):
        return self._name

    def setColor(self,c):
        self._color=c
    def getColor(self):
        return self._color

    def setWeight(self,w):
        self._weight=w
    def getWeight(self):
        return self._weight

    def display(self):
        print('Name:', self._name)
        print('Color:',self._color)
        print('Weight:',self._weight)

I have set attributes for one robot

    
        R=Robot()
        R.setName("Tom")
        R.setColor("Red")
        R.setWeight(55)

We can call attributes using the display function in the class 

    
        def display(self):
        print('Name:', self._name)
        print('Color:',self._color)
        print('Weight:',self._weight)

        R.display()

This is the output when we run the functions

    
        Name: Tom
        Color: Red
        Weight: 55
